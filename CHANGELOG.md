## [1.5.0] - 2026-06-27

### Added
- 对话可靠性增强：SSE 心跳保活、断线重连、事件恢复和结构化流式错误事件。
- 对话体验增强：消息引用回复、引用跳转、消息收藏、编辑重发、快捷键、输入历史回溯、平滑流式输出和打字光标。
- 对话数据能力：Token 实时统计、Token 预算控制、消息反馈、RAG 引用反馈和对话导出。
- Agent 能力增强：Agent 克隆、系统提示词变量预览、SubAgent 执行过程可观测和墙钟超时控制。
- RAG 能力增强：URL 文档增量同步、重复文档检测、查询参数推荐配置、查询改写和检索结果缓存大小限制。
- 工作流可靠性增强：条件分支默认路径校验、变量面板、节点超时配置和失败重试能力。
- API Key 能力增强：API Key 认证、Agent 作用域、调用限流和每日 Token 配额。
- 模型提供商扩展：新增 DeepSeek 与 Ollama 专用处理器，支持默认模型、模型列表拉取和连通性检查指定模型。
- MCP/Skill 能力增强：MCP 工具热更新、Skill 在线编辑和 Skill 依赖检查。
- 生产环境优化：数据库复合索引、pgvector HNSW 调优、Redis 缓存体系、MinIO 文件存储优化、消息列表虚拟滚动。

### Changed
- 版本号从 v1.0 升级到 v1.5.0，发布重点从基础功能可用升级为稳定性、可观测性和企业级可靠性治理。
- 对话服务进一步中间件化，统一处理初始化、消息构建、工具准备、追踪和流式输出。
- 前端 UI 样式统一迁移到 CSS 变量，增强暗色主题适配和组件一致性。
- 模型调用默认优先使用提供商配置中的默认模型，未指定时回退到处理器内置低成本模型。
- 工具选择、实体卡片、模型供应商管理等页面交互进一步简化。

### Fixed
- 修复流式模型调用失败时前端只看到连接关闭、无法识别错误原因的问题。
- 修复工具执行超时、消息跳转、消息关联和文件清理相关问题。
- 修复多个安全漏洞并完善权限控制。
- 修复暗色主题下部分组件颜色不一致问题。
- 修复版本配置 JSON 解析失败静默丢失排查困难的问题，增加日志提示。
- 消除部分 N+1 查询和内存聚合问题，提升数据库查询性能。

### Security
- API 工具执行链路加入 SSRF 防护、认证头构建和参数位置校验。
- API Key 增加限流、作用域校验和配额控制，降低滥用风险。
- 业务错误信息统一通过后端友好提示返回，前端优先展示业务错误信息。

### Removed
- 移除已合并的 CUSTOM 工具类型概念，统一纳入 API 工具体系。
