# LightBot Roadmap

> 轻量级现代化 Java AI Agent 平台
>
> Tech Stack: Spring Boot 3 + Spring AI + Vue 3

---

## 项目进度总览

| 阶段 | 状态 | 说明 |
|------|------|------|
| v0.1 MVP | ✅ 已完成 | Agent 基础能力 + 对话 + Tool Calling |
| v0.2 Agent + Tool 体系 | ✅ 已完成 | 自定义工具 + MCP 协议 + RAG 知识库 |
| v0.3 Workflow 引擎 | ✅ 已完成 | 可视化 DAG 编排 + 9 种节点 |
| v0.4 评测 + 可观测 | ✅ 已完成 | Eval 体系 + LLM Trace + Dashboard |
| v1.5 生产可用 | ✅ 已完成 | 基础能力 + 稳定性治理 + Phase 1 性能优化 |

---

## v0.1 MVP ✅

> 核心目标：跑通最小闭环，验证架构可行性

| 模块 | 功能 | 状态 |
|------|------|------|
| Chat | 单轮对话、多轮对话、流式输出 (SSE) | ✅ |
| Agent | 基础 Agent 定义（系统提示词 + 模型配置） | ✅ |
| Tool | 内置工具：HTTP 请求、当前时间 | ✅ |
| 模型 | OpenAI / 通义千问 接入 | ✅ |
| 存储 | 会话历史持久化 (PostgreSQL) | ✅ |

---

## v0.2 Agent + Tool 体系 ✅

## v1.5 生产可用 ✅

> 核心目标：完成基础能力闭环，并补齐稳定性、性能和可观测性治理。

### 已完成

| 模块 | 功能 | 状态 |
|------|------|------|
| 用户体系 | 注册 / 登录、个人信息、密码修改 | ✅ |

| 对话稳定性 | SSE 心跳、断线重连、结构化错误、重试处理 | ✅ |

## 暂不规划

| 特性 | 原因 |
|------|------|
| 微服务拆分 | 单体足够，过早拆分增加复杂度 |
| 自训练模型 | 专注平台能力，不涉及模型训练 |
| 区块链存证 | 场景有限，按需扩展 |
